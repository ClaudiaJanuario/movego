# MOVE GO - Branding Guidelines

## 🎨 Identità Visiva

### Il Nome: MOVE GO

**MOVE GO** è un gioco di parole che unisce:
- **MOVE** = Movimento, energia, dinamismo
- **GO** = Azione, partenza, viaggio
- **MG** = Le iniziali di **M**oto **G**uzzi

Il nome evoca:
- Il rombo del motore
- Lo spirito di avventura
- La tradizione che si muove verso il futuro
- L'invito a partire, esplorare, vivere

---

## 🦅 Logo Ufficiale

### Design
Il logo MOVE GO presenta un design **metallico industriale** che richiama l'heritage meccanico della Moto Guzzi:

- **Monogramma "MG"**: Le lettere M e G si intrecciano in un design tridimensionale
- **Colori**: Rosso Guzzi metallico (M) + Blu acciaio (G)
- **Texture**: Effetto metallo spazzolato con rivetti industriali
- **Tipografia**: "MOVE GO" in font sans-serif moderno, "MOTO GUZZI" come sottotitolo

### Simbolismo
- **Rosso**: Passione, velocità, cuore italiano
- **Blu acciaio**: Precisione meccanica, affidabilità, tradizione industriale
- **Rivetti**: Richiamo all'artigianalità e alla costruzione delle moto
- **Intreccio M-G**: Unione indissolubile tra Move Go e Moto Guzzi

### Versions
1. **Logo principale**: Full color su sfondo scuro
2. **Logo compatto**: Solo monogramma MG metallico
3. **Logo monocromatico**: Versione bianca o nera
4. **Favicon**: Monogramma MG semplificato

### Varianti Logo

| Variante | Uso |
|----------|-----|
| **Primary** | Header, materiali ufficiali |
| **Horizontal** | Banner, social cover |
| **Icon** | Favicon, app icon |
| **Monochrome** | Stampa B/N, incisioni |
| **Reversed** | Sfondi scuri |

## 🖼️ Stile Fotografico

### Mood Board

| Categoria | Caratteristiche |
|-----------|-----------------|
| **Paesaggi** | Lago di Como, montagne, strade panoramiche |
| **Heritage** | Foto d'epoca, museo, officina storica |
| **Moto** | V7, Le Mans, modelli iconici, dettagli |
| **Persone** | Guzzisti autentici, artigiani, guide |
| **Atmosfera** | Alba/tramonto, luce calda, nostalgia moderna |

### Trattamento Immagini

```css
/* Filtro Heritage */
filter: sepia(10%) contrast(105%) saturate(95%);

/* Overlay Dinamico */
mix-blend-mode: multiply;
background: linear-gradient(to bottom, transparent, rgba(26,26,26,0.7));
```

### Do's and Don'ts

✅ **DO**:
- Immagini autentiche di Mandello
- Moto Guzzi reali (non stock generici)
- Persone vere, non modelli
- Luce naturale, atmosfera italiana
- Dettagli meccanici, texture metalliche

❌ **DON'T**:
- Stock photos generiche
- Moto di altri brand
- Filtri eccessivi
- Immagini troppo patinate
- Contenuti non correlati al territorio

## ⚡ Animazioni

### Principi

1. **Sottili**: Mai distrarre dal contenuto
2. **Purposeful**: Ogni animazione ha uno scopo
3. **Performanti**: 60fps, GPU-accelerated
4. **Coerenti**: Stesse curve e timing

### Specifiche

```css
/* Transizione Standard */
transition: all 0.3s ease;

/* Hover Cards */
transform: translateY(-5px);
box-shadow: 0 20px 40px rgba(0, 0, 0, 0.15);

/* Engine Rumble (hover speciale) */
@keyframes rumble {
  0%, 100% { transform: translate(0, 0); }
  25% { transform: translate(0.5px, 0.5px); }
  50% { transform: translate(-0.5px, -0.5px); }
  75% { transform: translate(0.5px, -0.5px); }
}
```
