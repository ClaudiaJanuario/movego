## 🚀 Quick Start

```bash
git clone https://github.com/ClaudiaJanuario/movego.git
cd movego
npm install
npm run dev
```

Apri [http://localhost:3000](http://localhost:3000)

## 🛠️ Stack

- **Next.js 14** + React 18 + TypeScript
- **TailwindCSS** + Framer Motion
- **Multilingua**: IT, EN, DE, ES, PT
